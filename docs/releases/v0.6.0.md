# StockSight v0.6.0

This release connects strategy research to execution and review.

## Highlights

- Point-in-time Swing backtests with next-open entry assumptions, transaction costs, MFE/MAE, and 5/10/20-day outcomes.
- Chronological holdout calibration with Brier Score, Brier Skill, ECE, matched sample sizes, and reliability labels.
- ATR- and structure-based trade plans with trigger ranges, structural stops, two targets, account risk budgets, and lot-aware sizing.
- A persistent candidate → triggered → holding → exited → reviewed lifecycle.
- Actual fill and exit confirmation, automatic stop/second-target handling, P&L, R multiple, holding days, review grades, and an audit trail.
- Snapshot-safe rendering: replay can display lifecycle state without mutating the live ledger.
- Updated README documentation and report screenshots.

## Example

```bash
python scripts/report.py 002346 --mode detailed --strategy swing \
  --account-size 100000 \
  --lifecycle-file portfolios/trades.json \
  --html --out reports/002346-live.html
```

Holding state requires an actual `--fill-price`. StockSight does not treat a signal as a completed trade.
